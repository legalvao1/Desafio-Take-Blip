# Desafio Take Blip 🏆

-   Chatbot com fluxo conversacional com API intermediária integrando a Api do Github para visualização de repositórios específicos.

## Pre requisitos para rodar o projeto  ❗

-   Visual Studio Code
-   Node.js

## Tecnologias Implementadas  ⚙

-   Node.js
-   Express
-   Axios
-   Heroku Cloud
